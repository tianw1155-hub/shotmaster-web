# ShotMaster 阶段 3.3 组件收敛 + 令牌清理 设计规格

- **日期**：2026-07-04
- **状态**：待审阅
- **分支**：`phase3.3-cleanup`
- **前置**：Phase 1–3.2 已合并 `master`（`cbc1462`）。
- **关联**：Phase 3 ③ 子项目（组件收敛 + 令牌清理）。

## 1. 背景与目标

Phase 1–3.2 完成了功能建设，但积累了技术债：3 个组件在多页重复（HeroBack 3x、PlanSections 2x、ChapterHeader）、45 处兼容旧令牌名（`primary/mint/sun/sky/grape`）仍散布在 src/（兼容映射是 Phase 2 的临时措施）、以及多个 Minor（Badge 缺 `ink-muted`、deriveActive/deriveSection 重复、hairline hex 不对齐、默认头像 emoji、未用 import）。

**目标**：
1. 提取 3 个共享组件（HeroBack / PlanSections / ChapterHeader），消除重复。
2. 迁移全部 45 处兼容令牌 → 新名，移除 tailwind 兼容映射。
3. 修复积累的 Minor（Badge ink-muted、deriveActive 去重、hairline hex、头像、未用 import）。
4. 功能不回归。

## 2. 共享组件 API

### HeroBack
```tsx
// src/components/ui/HeroBack.tsx
interface HeroBackProps { onClick: () => void; className?: string; }
// 渲染：absolute top-4 left-4 w-10 h-10 rounded-full bg-surface-card/80 backdrop-blur
//       flex items-center justify-center shadow-elevated hover:bg-surface-card
//       ChevronLeft (lucide, stroke 1.25)
```
替换：LevelDetail:285、Gallery:597、Learn:261 的悬浮返回按钮。

### PlanSections
```tsx
// src/components/lesson/PlanSections.tsx
interface PlanSectionsProps {
  plan: ShootingPlan | null;
  isAnalyzing: boolean;
  imageId: string;
  onLike: (dim: ShootingPlanDimension) => void;
  onDislike: (dim: ShootingPlanDimension) => void;
  getFeedback: (dim: ShootingPlanDimension) => { liked: boolean; disliked: boolean };
}
// 渲染：AI 拍摄计划折叠手风琴（6 sections: scene/lighting/composition/params/postProcessing/equipment）
//       每个 section 可展开/折叠 + 赞/踩按钮。token: accent/gold/ink-muted/章节色。
```
替换：LevelDetail:184 + Gallery:337 的 `planSections` 数组 + 渲染。

### ChapterHeader
```tsx
// src/components/ui/ChapterHeader.tsx
interface ChapterHeaderProps { name: string; color: string; completed: number; total: number; }
// 渲染：章节色点 + 名 + N/10 + 进度条
```

## 3. 兼容令牌迁移（45 处）

迁移表（同 Phase 3.1）：
| 旧 | 新 |
|---|---|
| `text/bg/border-primary` | `accent` |
| `text/bg-mint` | `ink-muted`（或 `chapter.composition`） |
| `text/bg-sun` | `gold` |
| `text/bg-sky` | `ink-muted`（或 `chapter.narrative`） |
| `text/bg-grape` | `ink-muted`（或 `chapter.color`） |

涉及文件：`ErrorBoundary.tsx`、`ProgressBar.tsx`（color map）、`Button.tsx`（mint variant）、`Badge.tsx`（color map）、`GameComponents.tsx`（text-primary/bg-mint/text-sun/bg-primary 等 ~15 处）、`InteractiveLesson.tsx`（bg-sky）、`Login.tsx`（bg-primary/focus-ring/text-primary）、`LevelDetail.tsx`（planSections colors）、`Gallery.tsx`（planSections colors）、`LevelMap.tsx`（若有残留）。

迁移后：移除 `tailwind.config.js` 的 `primary/mint/sun/sky/grape` 兼容映射。
