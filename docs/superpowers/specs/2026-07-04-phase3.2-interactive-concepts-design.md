## 1. 背景与目标

阶段 1 只做了"曝光补偿"一个可交互概念。本 spec 新增 4 个 CSS 可行概念（白平衡 / ISO / 光圈·景深 / 焦距），复用 `InteractiveLesson` 的"拖动 → 实时图像 → 讲解 → 目标反馈"模式，让"动手学摄影"覆盖核心参数。

**目标**：
1. 4 个 concept config + filter 函数（`lib/filters.ts` 扩展）。
2. `InteractiveLesson` 小幅扩展以支持 transform / 分层 / 噪点 overlay。
3. Learn "动手练习" hub 集成（列出 5 个概念入口）。

## 3. InteractiveLesson 扩展（最小）

阶段 1 的 `ParamMap = (value:number)=>string`（返回 CSS `filter` 字符串），`InteractiveLesson` 把它设到 `img.style.filter`。为支持 4 个概念的真实效果，做 3 处最小扩展：

1. **ParamMap 返回 `CSSProperties`**（非纯 filter 字符串）：`type ParamMap = (value:number) => CSSProperties`。支持 `filter` + `transform`（焦距用 `transform: scale()`）。`InteractiveLesson` 把返回值整体设到 `img.style`（`style={concept.filter(value)}`）。曝光 concept 改为 `{ filter: 'brightness() saturate()' }`（兼容）。
2. **可选 `layers`**（ConceptConfig 加 `layers?: boolean`）：为 true 时 `InteractiveLesson` 渲染 2 个 img 层——背景层（应用 filter，如 `blur()`）+ 前景层（清晰，绝对定位 inset，可小偏移）。光圈用此（背景虚化、前景清晰）。
3. **可选 `overlay`**（ConceptConfig 加 `overlay?: (value:number) => { opacity: number } | null`）：返回非 null 时 `InteractiveLesson` 渲染一个噪点 overlay 元素（SVG `feTurbulence` 背景的 div，`mix-blend-mode: overlay`，`opacity` 由返回值控制）。ISO 用此（高 ISO 噪点）。

`exposure` concept（阶段 1）的 `filter` 改为返回 `{ filter: '...' }`（CSSProperties），兼容新签名。

## 4. 4 个滤镜映射

### 白平衡 WB（`concepts/wb.ts`）
- param: 色温 K，2000–10000，default 5500，step 100。
- filter（CSSProperties）: `t = (K-5500)/4500`；`sepia = max(0, -t*0.5)`；`hue-rotate = t*50deg`；`saturate = 1 + |t|*0.15`。→ `{ filter: 'sepia(s) hue-rotate(hdeg) saturate(a)' }`。
- target: 5500（日光），hitTolerance: 300。
- captions: <4500 暖偏黄（under）/ 4500–6500 正常（ok）/ >6500 冷偏蓝（over）。
- readout: `{value: K+'K', label: '色温'}`。
- image: 室内/人像场景（暖冷敏感）。

### ISO（`concepts/iso.ts`）
- param: ISO 100–6400，default 400，step 50。
- filter: `t = (iso-100)/6300`；`brightness = 1 + t*0.35`；`contrast = 1 - t*0.12`。→ `{ filter: 'brightness(b) contrast(c)' }`。
- overlay: `{ opacity: t*0.45 }`（噪点随 ISO 升）。
- target: 400（干净），hitTolerance: 150。
- captions: <250 低感光度·干净（ok）/ 250–1600 正常（ok）/ >1600 高感光度·噪点多（over）。
- readout: `{value: 'ISO '+iso, label: '感光度'}`。
- image: 低光场景（ISO 敏感）。

### 光圈·景深 Aperture（`concepts/aperture.ts`）
- param: f-stop 1.4–16，default 5.6，step 0.1。
- layers: true。背景层 filter: `t = (16-f)/(16-1.4)`；`blur = t*8px`。→ 背景层 `{ filter: 'blur(bpx)' }`；前景层无 filter。
- target: 2.8（浅景深），hitTolerance: 0.6。
- captions: <2.8 浅景深·背景虚化（ok）/ 2.8–8 正常（ok）/ >8 深景深·全清晰（over）。
- readout: `{value: 'f/'+f, label: '光圈'}`。
- image: 有明显前后景的场景（人像/静物）。

### 焦距 Focal（`concepts/focal.ts`）
- param: mm 24–200，default 50，step 1。
- filter（CSSProperties）: `t = (mm-24)/176`；`scale = 1 + t*0.8`。→ `{ transform: 'scale(s)' }`；容器 `overflow: hidden`（裁切）。
- target: 50（正常），hitTolerance: 15。
- captions: <35 广角·透视夸张（under）/ 35–85 正常（ok）/ >85 长焦·空间压缩（over）。
- readout: `{value: mm+'mm', label: '焦距'}`。
- image: 有纵深/远近元素的风景。

## 5. concept config schema（扩展）

```ts
interface ConceptConfig {
  key: string; title: string; kicker: string;
  image: { src: string; alt: string };
  param: { name: string; min: number; max: number; step: number; unit: string; default: number };
  filter: ParamMap;                       // (value) => CSSProperties
  readout: (v: number) => { value: string; label: string };
  target: number; hitTolerance: number;
  captions: { range: [number, number]; text: string; level: CaptionLevel }[];
  hitToast: string; nextConcepts: string[];
  layers?: boolean;                       // 新增：aperture（2 层）
  overlay?: (v: number) => { opacity: number } | null;  // 新增：ISO（噪点）
}
```

## 8. 边界与测试

- `lib/filters` 4 个 ParamMap 单测（输入→输出 CSSProperties/scale/blur/opacity 正确）。
- `InteractiveLesson` 扩展：layers 渲染 2 img；overlay 渲染噪点 div；transform 应用；exposure 兼容。
- 噪点 overlay 性能：SVG turbulence 是静态背景（`background-image` 一次生成），opacity 变化不触发重绘 turbulence——仅改 opacity（GPU）。reduced-motion 不影响（无动画）。
- 拖拽 filter/transform 走 GPU（transform/opacity/filter）。
- Learn hub：5 概念入口渲染 + 打开 InteractiveLesson。

## 9. 验收

1. 4 个 concept 可拖动交互：WB 暖冷、ISO 亮度+噪点、光圈背景虚化、焦距缩放裁切——实时、命中目标有反馈。
2. `InteractiveLesson` 扩展不破坏曝光 concept（兼容）。
3. Learn hub 列 5 概念 + 打开。
4. `lib/filters` 单测过；`npm run check`/`build` 绿；reduced-motion 不受影响。
